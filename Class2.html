INLS 161

Tools for information literacy is a class that concentrates on the unique tools and concepts that will allow me to communicate information to other users. The start of the class is focused on the basic building blocks needed for the Internet because it is important to have a good foundation before beginning to create well-developed and useful information for users. By the end of the class, we will begin to explore and design projectsv in other databases using the fundamental tools learned at the beginning. My personal goal in taking this class was to better understand how the Internet functions and to be able to better use different tools to create new designs in the future for my career or any other necessary time. The Internet is growing and developing rapidly so I believe it is important to learn as much as possible about the technology future generations will need to know.
